# Dockerized starter template for Laravel + Vite Vuejs project.

## Stack includes
* php-fpm
* Node.js
* MySQL
* Nginx

## Initial installation

1. Clone this repository.

```bash
$ git clone https://github.com/MaxmudovIsmoil/AiSoft-purchase.git
```

2. Running the initial installation.

```bash
$ cd AiSoft-purchase
$ make init
```
